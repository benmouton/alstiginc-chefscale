SCALING MATH LOGIC (in lib/scaling.ts):

function scaleAmount(originalAmount: number, originalServings: number, targetServings: number, unit: string, isScalable: boolean): { amount: number, unit: string, display: string }

The function should:
1. Calculate: scaledAmount = originalAmount × (targetServings / originalServings)
2. Apply unit conversions if amount crosses thresholds
3. Round appropriately (whole numbers for "each" items, nearest ¼ for cups, nearest 0.5 for tbsp/tsp)
4. Format display string with fractions where appropriate
5. Return non-scalable items unchanged
